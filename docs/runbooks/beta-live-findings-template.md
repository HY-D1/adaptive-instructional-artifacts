- **P0**: Data loss, security incident, total outage, crash

### Category Legend

- **Auth**: Login, signup, resume, session
- **Hint**: Hint request, loading, content quality, ladder progression
- **Textbook**: Save-to-notes, My Textbook, note formatting
- **Content**: Problem description, Learn tab, Examples, Common Mistakes
- **UX**: Navigation, layout, discoverability, feedback messages
- **Performance**: Load time, latency, timeout
