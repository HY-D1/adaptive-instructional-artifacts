# Static Textbook Content

Pre-processed textbook content for student learning.
These files are deployed with the app.

## Files

- `concept-map.json`: Maps concept IDs to PDF chunk IDs and metadata
- `textbook-manifest.json`: Index metadata including source documents
- `chunks-metadata.json`: Per-document chunk statistics
- `concepts/*.md`: Extracted content per concept (flat structure)
- `concepts/{docId}/*.md`: Extracted content per concept (helper-exported structure)

## Helper Integration

This content is generated by the `algl-pdf-helper` project.

### Export Command

```bash
# In helper repo: generate the index
python -m algl_pdf_helper index ./raw_pdf/murachs-mysql-3rd-edition.pdf \
  --output-dir /tmp/helper_index_out \
  --concepts-config ./concepts.yaml \
  --use-aliases

# Export to adaptive app
python -m algl_pdf_helper export /tmp/helper_index_out \
  --output-dir /ABSOLUTE/PATH/TO/adaptive-repo/apps/web/public/textbook-static \
  --merge
```

### Target Path

```
adaptive-repo/apps/web/public/textbook-static/
├── concept-map.json
├── textbook-manifest.json
├── chunks-metadata.json
└── concepts/
    └── {docId}/
        └── {conceptId}.md
```

### Concept ID Namespacing

The helper exports namespaced concept IDs: `{docId}/{conceptId}`

Examples:
- `murachs-mysql-3rd-edition/select-basic`
- `murachs-mysql-3rd-edition/where-clause`

The adaptive app resolves these to plain concept IDs:
- `murachs-mysql-3rd-edition/select-basic` → `select-basic`

This allows the helper to export multiple textbooks without ID collisions,
while the adaptive app maintains simple concept IDs for URLs and lookups.

### Resolution Strategy

When the adaptive app looks up a concept:

1. **Exact match**: Try the ID as-is (for already-canonical IDs)
2. **Namespace stripping**: If ID contains `/`, extract the suffix
3. **Suffix match**: Try the suffix against concept-map.json keys

Example resolution table:

| Helper Concept ID | Plain Concept ID | Resolution | Markdown Path Tried |
|-------------------|------------------|------------|---------------------|
| `select-basic` | `select-basic` | Exact match | `concepts/select-basic.md` |
| `murachs/select-basic` | `select-basic` | Suffix match | `concepts/murachs/select-basic.md` → `concepts/select-basic.md` |
| `select-basic` (in map as `murachs/select-basic`) | `murachs/select-basic` | Exact match | `concepts/murachs/select-basic.md` |

### Supporting Multiple Textbooks

To add a second textbook:

```bash
# Index the second PDF
python -m algl_pdf_helper index ./raw_pdf/another-textbook.pdf \
  --output-dir /tmp/helper_index_out2 \
  --concepts-config ./concepts.yaml \
  --use-aliases

# Export with merge to add to existing content
python -m algl_pdf_helper export /tmp/helper_index_out2 \
  --output-dir /ABSOLUTE/PATH/TO/adaptive-repo/apps/web/public/textbook-static \
  --merge
```

The `--merge` flag preserves existing concepts and adds new ones.
