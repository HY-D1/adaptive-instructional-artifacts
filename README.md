# Adaptive Instructional Artifacts (SQL)

This repo contains:

- Codebase notes (SQLBeyond / HintWise)
- Literature extraction + synthesis
- Prototype policies for “adaptive textbook” generation from traces

## Structure

- codebases/   (snapshots, pointers, integration notes)
- reviews/     (SQLBeyond + HintWise technical reviews)
- lit/         (papers + extraction table)
- synthesis/   (weekly syntheses)
- templates/   (note + extraction templates)
